**1) Observe o trecho de código abaixo: int INDICE = 13, SOMA = 0, K = 0;
Enquanto K < INDICE faça { K = K + 1; SOMA = SOMA + K; }
Imprimir(SOMA);
Ao final do processamento, qual será o valor da variável SOMA?**

**R:** Inicialmente temos o valor de K, sendo igual a 0, temos a INDECE sendo igual a 13 e SOMA igual a 0. Nesse cenário, quando se aplica o loop, (provavelmente um for) fazemos com o que o valor de K e SOMA sempre se atualizem, assim k será soma do seu valor com 1, enquanto a SOMA representa o seu valor atual adicionado ao valor de K.
